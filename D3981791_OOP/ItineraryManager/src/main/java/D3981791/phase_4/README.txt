You will need to change the Project Run settings to change the class where the main() method is found:

1. Right-click on the project name ('ItineraryManager') and select "Properties".

2. Click the "Run" category on the right-side panel of the dialog box that appears.

3. Click the "Browse..." button on the left, and the select and class that contains the main() method, then press the "Select Main Class" button.

4. Press the "OK" button to confirm the project's properties changes.
